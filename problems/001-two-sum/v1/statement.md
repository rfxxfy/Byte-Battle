## Условие

Дан массив из `N` целых чисел и число `target`. Найдите два индекса `i` и `j` (i < j), такие что `a[i] + a[j] = target`.

Гарантируется, что решение существует и оно единственное.

## Формат ввода

Первая строка — число `N`.

Вторая строка — `N` целых чисел через пробел.

Третья строка — число `target`.

## Формат вывода

Выведите два индекса (0-based) через пробел: `i j`, где `i < j`.

## Пример

**Ввод:**
```
4
2 7 11 15
9
```

**Вывод:**
```
0 1
```

*Объяснение: a[0] + a[1] = 2 + 7 = 9*
