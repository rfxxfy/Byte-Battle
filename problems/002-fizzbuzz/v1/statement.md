## Условие

Выведите числа от `1` до `N`, заменяя некоторые из них:

- Кратные **3** → `Fizz`
- Кратные **5** → `Buzz`
- Кратные **и 3, и 5** → `FizzBuzz`

## Формат ввода

Одно число `N`.

## Формат вывода

`N` строк — по одному значению на строку.

## Пример

**Ввод:**
```
5
```

**Вывод:**
```
1
2
Fizz
4
Buzz
```
